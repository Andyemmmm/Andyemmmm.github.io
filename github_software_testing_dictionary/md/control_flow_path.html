控制流路径

控制流路径是程序执行过程中可能遍历的所有路径的图形表示。大多数表示有两种类型的块。控制通过一个入口块进入流程图，所有控制流通过一个出口块离开。

通过这种表示，我们还可以计算圈复杂度。它还使我们能够有效地验证可达性、与其他节点、边缘和循环的关系。
